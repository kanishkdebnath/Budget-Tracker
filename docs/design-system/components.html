Component library

Everything the six screens will assemble from. All swatches use the navy palette. Every money figure is tabular. All interactive surfaces obey the pill/16/20 shape language from §1.

Five button roles cover everything. Primary CTA is filled with the brand-light primary in dark. Apply on Recurring uses tonal (primary-container) for a softer feel.
