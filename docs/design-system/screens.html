v15: Recurring rebalanced — 3-dot overflow icon dropped from applied rows; every card now has exactly one element on the right.

Where gradients live in v13 (restraint > everywhere):
1 · Net band — diagonal navy gradient (#1B4561 → #143548 → #0F2A3D) + a soft top-left brand-light glow + inset top highlight + drop shadow. The hero element now reads as lit, not flat.
2 · FAB — three-stop diagonal (#2A5E80 → #1B4561 → #143548) + inner top highlight + a navy-tinted drop shadow. Reads as a tactile pill.
3 · Filled / tonal buttons — 180° gradient (lighter top → tone bottom) + inset highlight. Catch the eye but not flashy.
4 · Card surfaces (txn, group, recurring, narrative, settings-group) — every card switched from flat --surface to a subtle #131A20 → #0F1619 top-light. ~3% lift at the top of each card.
5 · Bottom-nav active pill — same navy gradient as tonal button, with inset highlight.
6 · Banners — info gets the navy gradient, amber-highlight gets a warm #7B4520 → #5C3013 gradient. Inset highlight on both.
7 · Phone background — radial top-light rgba(28, 75, 105, 0.35) ellipse from (50%, -10%), fades to base black. Subtle brand-tinted atmospheric wash.
8 · Applied recurring — green tint became a vertical gradient (10% → 4% opacity). Applied check circle uses a green diagonal gradient for a tiny "polished button" feel.

Restraint: Plan/Actual numbers, ta-rows, txn-rows, settings-rows, chips, deltas — kept flat. Gradients are reserved for surfaces and interactive containers, not for data.
